# Diyestetik

Multi-tenant mimaride çalışan, estetik klinikler için tasarlanmış kurumsal web platformu.

## Tech Stack
- Next.js + TypeScript
- Supabase

## Özellikler
- Domain bazlı tenant yönetimi
- Klinik ayarları ve içerik yönetimi
- Kullanıcı davet ve yetkilendirme
- Modern ve responsive arayüz

Detaylı bilgi ve işbirliği için iletişime geçin.
