# Diyestetik

**🇹🇷 Türkçe**

Multi-tenant mimaride çalışan, estetik klinikler için tasarlanmış kurumsal web platformu.

## Tech Stack
- Next.js + TypeScript
- Supabase

## Özellikler
- Domain bazlı tenant yönetimi
- Klinik ayarları ve içerik yönetimi
- Kullanıcı davet ve yetkilendirme
- Modern ve responsive arayüz

---

**🇬🇧 English**

An institutional web platform designed for aesthetic clinics, operating in a multi-tenant architecture.

## Tech Stack
- Next.js + TypeScript
- Supabase

## Features
- Domain-based tenant management
- Clinic settings and content management
- User invitation and authorization
- Modern and responsive interface
